# 跑道特征检测系统 — 算法设计方案

## 一、任务目标

在飞机进近、着陆及跑道滑行阶段，检测跑道关键特征（角点、边线、中线），输出检测不确定性及跑道相对位姿估计。RTX 3090 上每帧推理时间 < 60ms。

## 二、总体架构

三阶段流水线：

1. **先验生成器**：利用 GPS/IMU 位姿和已知跑道几何信息，将跑道角点、边线、消影点投影到图像，渲染为 8 通道先验图
2. **视觉检测网络**：11 通道输入（RGB 3ch + 先验 8ch）→ EfficientNet-B0 + FPN → 三角点/线距离场/位姿三头预测 → 可微几何一致性层精化
3. **EKF 时序融合**：25 维状态向量，融合网络输出、GPS/IMU 测量、线几何约束，处理远场/近场模式切换

## 三、核心设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 检测范式 | 关键点 + 参数化线段联合检测 | 角点和线段互相校验，支持远/近场 |
| 先验融合方式 | 早期融合（先验图作为网络输入通道） | 网络隐式学习纠正先验偏差 |
| 不确定性估计 | 双头输出（Aleatoric + Epistemic） | 完整概率输出，与 EKF 协方差对齐 |
| 骨干网络 | EfficientNet-B0 + FPN | 精度-速度最佳平衡（~12ms TensorRT） |
| 训练策略 | 自监督为主（GPS 投影伪标签 + 几何约束自监督） | 无需人工标注，可扩展 |
| 核心创新 | 可微几何一致性层（展开 LM 优化） | 保证输出几何自洽，可端到端训练 |

## 四、各模块详细设计

### 4.1 先验生成器

基于现有 `correct_projection.py` 的坐标变换链：
- GPS → ECEF → NED → Body → Camera → Image
- 预计算跑道角点 ECEF 坐标和跑道坐标系轴向量

渲染 8 通道先验图（480×270 分辨率）：
- Ch 0-3：4 个角点的高斯热力图（σ=5px）
- Ch 4-6：左边界、右边界、中线的有符号距离场（归一化到图像对角线）
- Ch 7：消影点高斯热力图（由 GPS 位姿 + 相机内参解析计算）

CPU 渲染耗时 < 2ms。

### 4.2 视觉检测网络

**输入规格：** 11 通道（RGB 3ch + 先验 8ch），480×270 分辨率（从 1920×1080 双线性下采样）

**骨干网络：**

```
EfficientNet-B0（前 4 个 stage，总 stride=8）

stem (conv 3×3, stride=2)     → 240×135×32
MBConv1 (stride=1)             → 240×135×16
MBConv6 ×2 (stride=2)          → 120×68×24
MBConv6 ×2 (stride=2)          → 60×34×40
MBConv6 ×2 (stride=1)          → 60×34×80  ← 1/8 分辨率

FPN:
  P8 (60×34×80) → 上采样 2× + 1×1conv 融合 P4 → P4_out (60×34×128)
  P4_out → 上采样 2× → P_out (120×68×64)  ← 1/4 分辨率
```

**三个预测头：**

| 头 | 结构 | 输出通道 | 内容 |
|----|------|---------|------|
| 角点头 | Conv3×3→BN→ReLU→Conv1×1 | 8ch | 4个角点热力图 + 4个 log 方差图 |
| 线头 | Conv3×3→BN→ReLU→Conv1×1 | 6ch | 3条线距离场 + 3个 log 方差图 |
| 位姿头 | GAP→FC256→ReLU→FC128 | 12ch | 轴角R(3ch) + 平移t(3ch) + R方差(3ch) + t方差(3ch) |

**角点提取：** 热力图 → soft-argmax 可微峰值提取（亚像素精度）

**线参数提取：** 距离场 d=0 等值线 → RANSAC 直线拟合 → 归一化线参数 (a, b, c)，其中 a²+b²=1

**消影点计算：** 从三条线参数解析计算（两两交点，加权最小二乘解），不需要独立预测头

### 4.3 可微几何一致性层（核心创新）

**问题：** 网络三个头的原始输出互不保证几何一致性（角点不一定在相邻两线交点，三条线不一定交于同一点，位姿投影不一定与角点一致）。

**方案：** 将一致性约束表述为加权非线性最小二乘，通过展开优化（3-5 步 LM 迭代）精化输出。整个优化过程可微，梯度可反传到网络参数。

**隐变量（23 维）：**

| 变量 | 维度 | 含义 |
|------|------|------|
| ĉₖ | 8 (4×2) | 精化后的角点像素坐标 |
| l̂ₘ | 9 (3×3) | 精化后三条线的归一化参数 (a, b, c) |
| R̂, t̂ | 6 | 精化后的位姿（轴角 + 平移） |

**能量函数：** E = E_data + E_consistency + E_prior

**数据项 E_data（锚定网络原始输出）：**

```
E_data = Σₖ ||ĉₖ - cₖ_raw||² / σ²ₖ     ← 角点偏差（不确定性加权）
       + Σₘ mean_pixel ||d̂ₘ(p) - dₘ_raw(p)||² / τ²ₘ  ← 线距离场偏差
       + ||log(R̂ᵀR_raw)||² / σ²ᵣ + ||t̂ - t_raw||² / σ²ₜ  ← 位姿偏差
```

**几何一致性项 E_consistency：**

```
E_consistency = 
  α₁ · ||三线消影点互差||₁                      ← 消影点一致性
+ α₂ · Σₖ ||四个角点与对应两线交点的距离||₁       ← 角线一致性
+ α₃ · ||入口边方向 - 出口边方向||₁              ← 平行约束
+ α₄ · ||Proj(R̂,t̂,K,3D_cornerₖ) - ĉₖ||₁      ← 重投影一致性
```

**先验项 E_prior：**

```
E_prior = β₁ · ||R̂ - R_gps||² / σ²_gps_r + β₂ · ||t̂ - t_gps||² / σ²_gps_t
```

**求解：展开 LM 优化**

```
for i = 1 to K (K=3~5):
    θ_{i+1} = θ_i - (JᵀWJ + λI)⁻¹ JᵀW r(θ_i)
```

- J：雅可比矩阵（23 变量 × ~30 残差），通过 PyTorch autograd 获得
- W：不确定性权重对角阵（由网络方差输出决定）
- λ：LM 阻尼因子（可学习参数）

梯度通过每次迭代反传到网络参数。

### 4.4 EKF 时序融合

**状态向量（25 维）：**

```
x = [lat, lon, alt,              ← 飞机 WGS84 位置 (3)
     v_n, v_e, v_d,              ← 飞机 NED 速度 (3)
     roll, pitch, yaw,           ← 飞机姿态角 (3)
     ĉ₁_u, ĉ₁_v, ..., ĉ₄_u, ĉ₄_v, ← 4 角点图像坐标 (8)
     vp_u, vp_v,                 ← 消影点图像坐标 (2)
     ba_x, ba_y, ba_z,           ← IMU 加速度 bias (3)
     bg_x, bg_y, bg_z]           ← IMU 陀螺仪 bias (3)
```

**预测步骤（IMU 驱动，Δt ≈ 33ms @ 30fps）：**

- 姿态传播：中值积分，q_{k+1} = q_k ⊗ exp((ω_k - b_g) · Δt)
- 位置传播：欧拉积分
- 特征传播：恒速模型，ĉ_{k+1} = ĉ_k + v_c · Δt，过程噪声 σ²=10px²

**三项观测更新：**

| 观测 | 维度 | 模型 | 噪声来源 |
|------|------|------|---------|
| GPS | 3 | 线性 | GPS 精度 (~1m, 1m, 3m) |
| 网络输出 | 20 | 非线性 h_net(x) | 网络预测方差 R_net |
| 线约束 | 2（等效） | vp 从线交点提取 | 线方差传播 |

**计算复杂度：** 25 维状态，EKF 单步 < 0.3ms。

### 4.5 远场/近场模式切换

**远场模式判定：** 可见角点数 ≥ 4 且高度 > 30m AGL
**近场模式判定：** 底部 2 角点不可见或高度 < 30m

**切换逻辑：**

- 远→近：对底部角点观测做渐变去激活（sigmoid 权重，5-10 帧过渡）
- 近→远：反向激活
- 近场位姿求解：远端 2 角点 + 消影点方向 + 已知跑道宽度 → 最小二乘位姿

**过渡函数：**

```
w_bottom(t) = sigmoid((h(t) - h_threshold) / δh)
```

### 4.6 损失函数

```
L_total = L_data + λ_geo·L_geo + λ_smooth·L_smooth + λ_unc·L_unc
```

**监督损失 L_data（与伪标签对齐）：**

| 子损失 | 公式 | 用途 |
|--------|------|------|
| L_corner_focal | Focal loss（α=2, β=4）| 角点热力图 |
| L_line_L1 | 逐像素 L1（仅在有真值线段区域） | 线距离场 |
| L_pose_huber | Huber loss | 位姿回归 |

**自监督几何损失 L_geo：**

```
L_geo = ||vp(l̂_l, l̂_r) - vp(l̂_l, l̂_c)||₁       ← 消影点
      + Σₖ ||ĉₖ - intersect(l̂_e1ₖ, l̂_e2ₖ)||₁   ← 角线一致性
      + ||Proj(R̂,t̂,K,Pₖ) - ĉₖ||₁              ← 重投影
      + ||dir(l̂_entrance) - dir(l̂_exit)||₁     ← 平行约束
```

**时序平滑损失 L_smooth：**

```
L_smooth = ||ĉₖ(t) - ĉₖ(t-1)||² / σ²ₖ + ||h(t) - h(t-1)||²
```

**不确定性损失 L_unc：**

```
L_unc = Σ_{各头} [log(σ²) + ground_truth_error² / σ²]   （负对数似然）
```

### 4.7 训练流程（三阶段）

| 阶段 | 数据 | 损失 | 目的 |
|------|------|------|------|
| Phase 1 | 仿真数据（全标注） | L_total | 学到基本检测能力 |
| Phase 2 | 仿真 + 真实混合 | L_total(仿真) + (L_geo+L_smooth)(真实) | Sim-to-Real 适应 |
| Phase 3 | 仿真 + 真实 | L_total + MC Dropout epistemic loss | 不确定性标定 |

## 五、评测指标与基准

### 5.1 角点检测精度

| 指标 | 定义 | 目标 |
|------|------|------|
| NME | `‖ĉ - c_gt‖₂ / diag(图像)` × 100% | < 0.5% (≈ 11px) |
| PCK@5px | 角点落在真值 5px 半径内比例 | > 90% |
| 检测率 | 置信度 > τ 的角点比例 | > 95% |

### 5.2 线段检测精度

| 指标 | 定义 | 目标 |
|------|------|------|
| 角度误差 | 检测线与真值线夹角 | < 0.5° |
| 正交距离 | 到真值线的平均垂距（归一化） | < 0.3% diag |
| 端点误差 | 线段起止点与真值的平均距离 | < 15px |

### 5.3 位姿估计精度

| 指标 | 定义 | 远场(>30m) | 近场(<30m) |
|------|------|-----------|-----------|
| 平移误差 | `‖t̂ - t_gt‖₂` (m) | < 5m | < 2m |
| 旋转误差 | `‖log(R̂ᵀR_gt)‖₂` (°) | < 2° | < 1° |
| 重投影误差 | 投影角点与 GT 角点平均像素差 | < 5px | < 8px |

### 5.4 不确定性质量

| 指标 | 定义 |
|------|------|
| ECE | 期望校准误差（10 bin），|置信度 - 准确率| 的加权平均 |
| NLL | 负对数似然：`-log p(y_gt | μ̂, σ̂²)` |
| Sharpness | 预测方差的均值（同等校准下越小越好） |

### 5.5 鲁棒性指标

| 场景 | 条件 | 允许退化 |
|------|------|---------|
| 光照变化 | 日出/正午/黄昏/阴天 | NME < 1.0% |
| 天气 | 小雨/雾（仿真注入） | NME < 2.0% |
| 部分遮挡 | 云遮挡跑道局部 | 检测率 > 80% |
| 近场 | 高度 < 30m | 位姿误差 < 3m |

### 5.6 时序稳定性

```
角点抖动: δ_corner = std(ĉ(t) - ĉ(t-1)) / NME   目标 < 0.3
位姿抖动: δ_pose   = std(t̂(t) - t̂(t-1)) / 平移误差  目标 < 0.2
```

### 5.7 基准方法

| 基准 | 说明 | 目的 |
|------|------|------|
| 纯 GPS 投影 | 无视觉，直接 GPS+IMU 投影 | 衡量视觉的必要性 |
| 纯视觉 CenterNet | 仅角点头，无先验/无线/无几何约束 | 消融先验价值 |
| PnP+GPS（`correct_projection.py`） | GPS 投影直接做 PnP | 衡量学习部分的价值 |
| Ours w/o 几何层 | 完整网络但不跑几何优化 | 消融几何层价值 |
| Ours full | 完整方案 | 全部组件 |
